# Advent of Code 2025
This is my personal solution repository for Advent of Code 2025. 
The code is written entirely in c++.
There are a total of twelve different coding challenges, each with increasing levels of difficulty. 
Merry Christmas!

~ Sebastian Gluszak

## To build and executable for main run the following:

```clang++ -std=c++2c -Wall -Wextra -o main main.cpp src/DayOne.cpp```

## To run the main executable run the following:

```./main <day> <part>```

## Acceptable day and part combinations

- Day: 1 Part: 1
- Day: 1 Part: 2
